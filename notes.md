## Commands

* `set var val` - set var, warn if overwriting
* `list add var val` - add item to list
* `list remove var val` - remove item from list
* `list dedupe`
* `list remove-at i`
* `run` - run script in dsl
* `path cmd` - equrivalend to `list cmd PATH`
